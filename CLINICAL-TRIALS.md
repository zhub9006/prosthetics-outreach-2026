## Trial #1: ENABLE — Smart Socket Fit Assessment

| Field | Detail |
|-------|--------|
| **NCT ID** | [NCT05425810](https://clinicaltrials.gov/study/NCT05425810) |
| **Title** | R21: An Easy-to-use, iNtelligent, Affordable LinEr (ENABLE) System for Socket Fit Assessment |
| **Status** | 🟢 RECRUITING |
| **Sponsor** | North Carolina State University |
| **PI** | Ming Liu, Research Assistant Professor |
| **Phase** | N/A (Device Feasibility) |
| **Condition** | Lower Limb Amputation Below Knee |
| **Enrollment** | 3 (estimated) |
| **Start** | April 20, 2021 |
| **Completion** | Feb 2026 (estimated) |
| **FDA Regulated** | Yes (Device) |

### Summary
Developing a low-cost pressure-sensing pad integrated into a prosthetic liner that uses nanotube-based sensors with thermal dyes to visually display pressure distribution on the residual limb, helping prosthetists optimize socket fit without expensive equipment.

### Key Innovation
- Gallium-alloy-filled nanotubes embedded in standard liner material
- Color-change visual feedback — no complex electronics needed
- 3–4 visit protocol: socket construction → prototype validation → fit assessment

### Eligibility (Inclusion)
- Unilateral transtibial amputees
- ≥1 year prosthesis experience
- Current socket used ≥6 months without skin issues
- Can walk 3 mins without resting

## Trial #2: AI-Powered Robotic Hand — 3D-Printed Low-Cost Prosthetic

| Field | Detail |
|-------|--------|
| **NCT ID** | [NCT07437664](https://clinicaltrials.gov/study/NCT07437664) |
| **Title** | AI-Powered Robotic Hand for Grasping Objects Using 3D Scanning & Printing |
| **Status** | 🟡 ACTIVE NOT RECRUITING |
| **Sponsor** | Al-Nahrain University, Iraq |
| **PIs** | Amjed Abdulameer Hussein, Wajdi Sadik Aboud (Prof.) |
| **Phase** | N/A (Device Feasibility) |
| **Condition** | Transradial Amputation |
| **Enrollment** | 1 (actual) |
| **Location** | Baghdad, Iraq |

### Summary
A **low-cost, 3D-printed prosthetic hand** with AI-powered EMG signal classification. Uses 3D scanning to create custom sockets, then AI to interpret muscle signals for adaptive grasping — dramatically reducing cost vs. traditional prosthetic fabrication.

### Key Innovation
- 3D scanning + 3D printing for custom sockets at a fraction of traditional cost
- AI pattern recognition of EMG signals for intuitive, adaptive control
- Compares AI-driven vs. traditional rule-based control

### Eligibility (Inclusion)
- Unilateral transradial amputation
- Age 18–60
- Detectable EMG signals from residual limb

## Trial #3: ORTHOPUS — Low-Cost Prosthetics for LMI Countries

| Field | Detail |
|-------|--------|
| **NCT ID** | [NCT04803253](https://clinicaltrials.gov/study/NCT04803253) |
| **Title** | ORTHOPUS Prosthetic Solutions for Upper Limb Amputees in Nepal |
| **Status** | ⚪ UNKNOWN (was Recruiting) |
| **Sponsor** | Orthopus / Handicap International Nepal |
| **Phase** | N/A |
| **Condition** | Upper Limb Amputation |
| **Enrollment** | 14 (2 cohorts of 7) |

### Summary
Specifically designed for **low- and middle-income countries** by offering low-cost but high-quality prosthetic solutions where only 2 options are typically available:
- Cosmetic articulated hand
- Mechanical wrist
- Work hook
- Body-powered harness

Directly addresses supply and cost constraints in underserved regions.

### Key Innovation

- Extends grasping beyond passive aesthetic hands
- WHO GATE community reporting
